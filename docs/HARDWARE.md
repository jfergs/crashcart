# CrashCart Hardware

## Primary Platform

### Raspberry Pi Zero v1.3
Reasons:
- USB OTG gadget support
- Extremely low power
- Small footprint
- Widely available accessories

## Planned Accessories

- UART breakout cables
- USB OTG adapters
- USB Ethernet adapters
- OLED displays
- Compact enclosures
- Magnetic serial adapters

## Future Exploration

- Pi Zero 2 W
- CM4 variants
- RP2040 helper boards
