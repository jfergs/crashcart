# CrashCart Roadmap

## v0.1 - BootROM
Initial proof-of-concept release.

Goals:
- Bootable Pi Zero image
- USB Ethernet gadget
- SSH access
- Serial console support

---

## v0.2 - Console Cart
Focus on infrastructure console recovery.

Goals:
- UART toolkit
- Session logging
- Web serial console
- Adapter profiles

---

## v0.3 - NetRunner Cart
Networking and diagnostics release.

Goals:
- VLAN diagnostics
- mDNS testing
- Packet capture
- UniFi helper tools

---

## v0.4 - BootPak
Portable recovery media release.

Goals:
- ISO hosting
- Recovery partition
- Firmware vault
- Portable documentation

---

## v1.0 - CrashCart
Stable release.

Goals:
- Cartridge system
- Web dashboard
- Save-state configuration backups
- Full Ansible provisioning
