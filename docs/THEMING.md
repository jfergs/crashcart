# CrashCart Theming Guide

CrashCart uses retro gaming and service cartridge aesthetics.

## Themes

### Visual Themes
- CRT phosphor green
- Game Boy monochrome
- SNES gray/purple
- Pixel-art iconography

### Terminology
- Cartridge
- ROM
- SaveRAM
- Warp Zone
- BootPak
- Link Cable
- Continue?
- Safe Mode

## UI Philosophy

The retro theme should:
- enhance usability
- create a memorable identity
- remain restrained and readable

Avoid:
- excessive animations
- novelty over usability
- inaccessible color schemes
